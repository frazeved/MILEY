Monthly summary table comparing 2025 performance vs 2026 plan. Shows invoiced vs forecast vs PO issued for each month of the year plus a PO ISSUED category breakdown table.

API Endpoints
GET
/api/forecast
Returns monthly aggregates for Forecast & Budget page. Arrays of 12 values (Jan–Dec) for inv2025, forecast2026, po2026, inv2026 plus catBreakdown object.
?refresh=1 — bypass 10-min cache and re-fetch from Sheets
GET
/api/forecast-by-category
Returns all Tradestone rows (style, status, supplier, brand, category, qty, wholesale, invoiced, cancelDate, invDate) plus filter option lists.
?refresh=1 — bypass cache
GET
/api/map-overview
Returns all MAP ANTHRO 2026 rows with key fields (status, supplier, style, po, qty, boxQty, fob, wholesale, shipDate, cancelDate, arrival, brand, category, supInvoice, urbnTotal, airfreight, groundFreight…).
?refresh=1 — bypass cache
